Instructions

These simple steps will help you use the FajitaBoy correctly.

Step 1: Obtain a ROM

The first thing you'll need to do in order to use FajitaBoy is to obtain a game you wish to play, like Tetris for example. FajitaBoy can play most Game Boy and Game Boy Color games. Save the ROM file to your harddrive and remember where you put it.

Step 2: Loading the FajitaBoy applet

FajitaBoy is a signed Java applet. When loading the applet you will therefore get a popup asking for your trust in the unknown. Depending on your operating system, it might look something like this

Unfortunately, you'll need to accept this certificate in order to use FajitaBoy. If you didn't accept, and change your mind, you might need to restart your browser to get another chance.

Why does FajitaBoy need a certificate? Java applets are not allowed to read files from the users harddrive by default. FajitaBoy however needs to read the ROM files provided by the user in order to play games. The certificate allows FajitaBoy to read local files. If you are afraid we intend to abuse this right, feel free to check out our source code.

Step 3: Using the FajitaBoy applet

To start a singleplayer game, click the singleplayer button.

Next up you need to load the ROM file from your local harddrive. FajitaBoy accepts .gb, .gbc and .zip files. You can do this either by manually entering the path into the textfield...

by browsing for it.

When you have selected a file, click the Start Game button to continue.

The emulator will now initiate, and you can start playing the game.

By default you use the arrow keys for joypad controls, 'x' and 'z' for A and B buttons and Enter och Shift for Start and Select buttons. These settings can be changed, as you will see a few steps down.

Other useful keys are 'p' to toggle Pause, 'm' to toggle Mute and Esc to bring up or close the ingame menu.

Step 4: Using the ingame menu

The ingame menu is divided into three tabs - Game, Keys, Sound.

In the game tab FajitaBoy provides the possibility to save states, ie saving the game. You can later on load the state to resume the game as it was when you saved. You can also reset the current ROM or go back to the main menu and load a new one using this menu.

The Keys tab allows you to change your keybindings. To change a key use the mouse to click a button and then click a new key. The Reset button sets all keys to the default bindings.

If you have cookies enabled you can save your keybindings. These will then be loaded automatically the next time you use FajitaBoy.

In the Sound tab you can change the volume of the emulator. Setting it to zero will disable the audio which can give a slight boost to performance.
